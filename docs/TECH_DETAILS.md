# 📊 QuantAnalyzer Pro

A premium tool for analyzing your MT5/MT4 Strategy Tester reports with a focus on **Martingale Sequence Analysis**.

## 🚀 How to use
1. Open the `index.html` file in any modern web browser (Chrome, Edge, Firefox).
2. Export your **Strategy Tester Report** from MetaTrader as an **HTML** file.
3. **Drag and drop** the HTML file into the analyzer.
4. Instantly view your equity curve, drawdown, and detailed sequence "proof data".

## ✨ Key Features
- **Auto-Detection**: Automatically identifies the Deals table in your MT5 reports.
- **Martingale Logic**: Groups trades into sequences to show you exactly how deep your Martingale went.
- **Visual Insights**: Smooth equity curve chart and floating drawdown calculation.
- **Export**: Save your sequence analysis as a CSV for use in Excel or Google Sheets.

## 📁 Files
- `index.html`: The complete application (no installation needed).
- `martingale_analysis.csv`: (Generated after clicking Export)
