# Fuji Martingale Streak

A high-end, premium statistical analyzer for MetaTrader (MT4/MT5) Martingale strategies. 

![Aesthetics](https://img.shields.io/badge/Aesthetics-Premium-blueviolet)
![Stats](https://img.shields.io/badge/Stats-Empirical-success)

## 🚀 Overview
Fuji Martingale Streak is designed for quantitative traders who need to verify the robustness of their Martingale systems. It analyzes trade reports to identify loss clusters, calculate average frequencies, and visualize the true risk of your strategy.

### Key Features
- **Data-Driven Analysis**: Uses your actual trade win/loss ratio, not theoretical assumptions.
- **Visual Distribution**: Beautiful, high-end charts showing the frequency of every martingale depth.
- **Standalone Desktop App**: Compiled as a secure .exe for private use.
- **Premium Exports**: Generate professional PNG reports for sharing and documentation.

## 📂 Project Structure
- `src/`: Core application logic (HTML/JS and Python Wrapper).
- `tools/`: Build scripts and compilation tools.
- `docs/`: Documentation and assets.

## 🛠️ How to Build (.exe)
1. Ensure Python is installed.
2. Run `pip install pyside6 pyinstaller`.
3. Go to `tools/` and run `build_exe.bat`.
4. Your standalone app will appear in the `dist/` folder.
